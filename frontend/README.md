# Job Tracker Frontend

A React TypeScript frontend for a personal Job Tracker application.  
Users can sign up, log in, save jobs, track application status, and manage their saved job listings.

## Features

- User signup and login
- Protected routes
- Save job applications
- View saved jobs
- Update job status
- Delete saved jobs
- Responsive UI
- API integration with backend

## Tech Stack

- React
- TypeScript
- Vite
- React Router
- Axios
- CSS
